# 🏥 Healthcare Dashboard

An interactive and responsive Healthcare Dashboard built using HTML, CSS, and JavaScript to visualize patient data, appointments, and healthcare insights in a clear and efficient way.

## 🚀 Features

- 📊 Interactive charts and data visualization using Chart.js  
- 🧑‍⚕️ Patient data management system  
- 📅 Appointment tracking  
- 📈 Real-time data representation (static/mock data)  
- 🎨 Clean and user-friendly UI  
- 📱 Fully responsive design  

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Chart.js  
- Postman (for API testing)
